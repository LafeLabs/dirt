# [dirt](https://github.com/LafeLabs/dirt/)

![](qrcode.png)

A human front end for Python using [P5.js](https://p5js.org/) and [Trash Magic](https://trashrobot.net)(self-replicating self-hosted HTML/PHP). Draw glyphs on a screen, move your mouse, scroll your mouse wheel, or type in a text box, record an image from a camera, or speak into a micropohone, and Python will respond with whatever you tell it to respond with. dirt.html is a web app template which loads dirt.json, edits it based on human input, and sends it to dirt.py using bridge.php, and then dirt.py does whatever you tell it to and returns JSON data over the bridge which includes a base64 encoded image of whatever you told Python to generate, as well as html generated by python and sent back to the dirt.html. 

Dirt.html has a control pane on the left and a data pane on the right. 

data pane shows plots which are saved along with data which created a plot. User pushes button to save plots and steers around with text, mouse, glyphs, sound, and images to make the Python do things and return data plots which the user then uses to decide to save data.


dirt.ipynb is a notebook which feteches data sets from across the network and replicates them locally and pushes local data sets to data exchanges. this is the real magic of dirt. when we create networks of social notebooks which can all fetch and push data between each other, and that data is linked with the web front ends with human operators

this is to build a swarm intelligence across operators and data streams, all connected to Python on the back end, with the notebooks controlling data flow. Having data push and fetch be in a jupyter notebook keeps that from getting out of control like it could if this were all automated. 

Humans move data around with the notebooks in one browser and in another browser interact with data live via the real time interface which talks to the Python on their machine which can in turn see all the data on the network.

## Install code on linux:

```
sudo apt update
sudo apt install apache2 -y
sudo apt install php libapache2-mod-php -y
cd /var/www/html
sudo rm index.html
sudo apt-get install curl
sudo curl -o replicate-file-set.php https://raw.githubusercontent.com/LafeLabs/dirt/refs/heads/main/replicate-file-set.php
cd ..
sudo chmod -R 0777 *
cd html
php replicate-file-set.php
sudo chmod -R 0777 *
ln -s /var/www/html ~/Desktop/html
```
